# 💪 ALPHA FITNESS

**A strength & conditioning gym website — built for a modern, fully AC, professionally equipped training facility.**

🔗 **Live Site:** [www.alphafitnes.xyz](https://www.alphafitnes.xyz/)

---

## 📖 About

ALPHA FITNESS is the official website for a strength and conditioning gym offering personal training, a fully air-conditioned facility, modern equipment, and structured programs for fat loss, muscle gain, and strength training — all in a clean, hygienic environment.

This site serves as the gym's digital front door: giving potential members a clear view of services, trainers, pricing, and a way to get in touch.

## ✨ Features

- 🏋️ **Services overview** — strength training, fat loss & muscle gain programs
- 👨‍🏫 **Trainers page** — meet the coaching team
- 💰 **Pricing page** — membership plans at a glance
- 📩 **Contact page** — get in touch / inquire about membership
- ❄️ Fully AC facility, modern equipment, hygienic environment — highlighted throughout
- 📱 Responsive design for mobile and desktop

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 |
| Interactivity | Vanilla JavaScript |

No frameworks, no build step — plain, fast, and easy to deploy anywhere.

## 🚀 Getting Started

Since this is a static site, there's no build process required.

```bash
# Clone the repo
git clone https://github.com/Sampad7/ALPHA-FITNESS.git
cd ALPHA-FITNESS

# Open directly in your browser
# or serve locally, e.g.:
npx serve .
```

Then visit `http://localhost:3000` (or just open `index.html` directly).

## 🌍 Deployment

The live site is hosted at [alphafitnes.xyz](https://www.alphafitnes.xyz/). As a static site, it can be deployed to any static host — GitHub Pages, Netlify, Vercel, or traditional hosting.
